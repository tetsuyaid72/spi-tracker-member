PRD Frontend: SPI Member Tracker (Multi-User)1. Arsitektur Antarmuka (Role-Based UI)Aplikasi akan mendeteksi peran pengguna setelah login dan menampilkan dashboard yang berbeda:Role User (Deliman): Fokus pada pemetaan lokasi dan efisiensi tagging di lapangan.Role Admin: Fokus pada manajemen data, visualisasi agregat, dan kontrol pengguna.2. Fitur Utama FrontendA. Autentikasi & ProfilLogin/Register Page: Input email/nomor HP dan password.Role Identification: Status akun (Admin/User) terlihat di bagian profil.Session Management: Tetap login (Remember Me) untuk memudahkan penggunaan di jalan.B. Fitur Map & Heatmap (User)My Map View: Menampilkan marker toko yang hanya ditandai oleh user tersebut.Interactive Tagging: Tombol melayang (Floating Action Button) untuk simpan lokasi dengan satu ketukan.Personal Heatmap Toggle: Tombol untuk menyalakan/mematikan tampilan intensitas kunjungan pribadi.Offline Tracking: Kemampuan menandai lokasi meskipun sinyal lemah (tersimpan di cache hingga sinyal kembali).C. Fit का Dashboard (Admin)Global Heatmap: Peta yang menggabungkan data dari seluruh deliman untuk melihat kepadatan member secara total.User Management Table: Daftar semua deliman, jumlah toko yang mereka tandai, dan aktivitas terakhir.Data Filtering: Filter heatmap berdasarkan nama deliman, rentang waktu, atau wilayah tertentu.Export Tool: Tombol untuk mengunduh koordinat toko ke format CSV/Excel.3. Spesifikasi Teknis FrontendKomponenTeknologiAlasanFrameworkReact.js / Next.jsCepat, SEO friendly, dan ekosistem library peta yang luas.PetaLeaflet.jsSangat ringan untuk HP dengan spesifikasi rendah.StylingTailwind CSSMempermudah pembuatan desain mobile-first yang responsif.IconsLucide ReactIkon yang bersih dan ringan untuk navigasi.PWA SupportVite PWA PluginAgar aplikasi bisa di-install di layar utama HP tanpa melalui Play Store.4. Alur Pengguna (User Flow)User (Deliman)Login -> Lihat Peta -> Klik "Tandai Toko" -> Isi Nama Toko -> Simpan.Saat lupa, buka "Daftar Toko" atau cek "Heatmap" untuk navigasi visual.AdminLogin -> Dashboard Utama -> Lihat sebaran marker seluruh tim.Filter data berdasarkan Helper A atau Helper B untuk evaluasi rute.5. Desain UI (Wireframe Thinking)Header: Logo SPI, Search Bar (cari toko), dan Profil.Main Area: Peta fullscreen dengan kontrol zoom di sisi kanan.Footer (Mobile): Tab Navigasi (Peta, Daftar Toko, Pengaturan).Warna: Biru/Kuning (Identitas Indogrosir) dengan warna Heatmap (Hijau ke Merah).6. Rencana IterasiV1 (MVP): Login sederhana, tagging lokasi, dan heatmap pribadi.V2: Dashboard admin untuk melihat semua data dan filter user.V3: Fitur berbagi lokasi toko antar sesama rekan helper.
